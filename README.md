# Forkline

中文 Web 端 Git 可视化管理工具。无需安装 npm 依赖，使用 Node.js 内置 HTTP 服务启动。

## 功能

- GitKraken 风格提交图谱，全部分支模式下以 `main` / `master` 为主线。
- 本地分支、远端分支、工作区状态和未暂存更改查看。
- 文件路径树状归纳，支持同层级聚合与滚动查看。
- 工作区 Diff、历史提交 Diff、小窗口预览和最大化对照。
- 可拖拽左侧栏、右侧栏、底部变更面板、文件列表宽度和提交框高度。
- 支持抓取、拉取、推送、暂存全部、单文件与 Ctrl / Shift 多选暂存 / 取消暂存、丢弃未暂存或已暂存改动、创建提交和追加提交。
- 支持创建储藏、储藏所选文件、查看储藏 Diff、应用 / 弹出 / 删除储藏。
- 支持本地分支切换、远端分支签出为本地跟踪分支、保留 merge commit 的分支合并、新建/重命名/删除分支和创建 Tag。
- 支持修改提交信息：最新提交使用 amend，历史提交使用 rebase reword。

## 启动

```bat
start.cmd
```

打开：

```text
http://127.0.0.1:5177
```

默认显示示例数据。顶部输入本机 Git 仓库路径后点击 `打开`，即可读取真实仓库。

## 注意

真实仓库模式下的写入操作会先弹窗确认，再由本地 Node 服务执行 Git 命令。修改历史提交信息会重写相关 SHA，工作区不干净时会被阻止。
